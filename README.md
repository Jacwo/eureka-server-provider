# eureka-server-provider
服务提供者

## 安装要求
* git 2.21.0
* maven 3.6.1
* JDK 1.8

## 快速开始
*  启动服务注册中心 参考  https://github.com/Jacwo/eureka-server
* ``git clone https://github.com/Jacwo/eureka-server-provider.git `` 

* ``cd eureka-server-provider ``
 
* ``mvn spring-boot:run ``

## 测试

* 打开浏览器输入 http://localhost:8761/hello 查看
